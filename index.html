System Architecture

A GitHub Action runs fetch-news.js daily at 6 AM UTC. It queries GNews API for each company, deduplicates against existing articles, auto-categorizes them, and commits updated data files to the GitHub repository. GitHub Pages serves the static site. Read/unread state and column order are stored in localStorage.
